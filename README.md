# CTD-1D Game Project: Fablab Adventures 

Fablab Adventures is a 2 dimensional game made with the Turtle Library. 
Libraries used:
- Turtle
- Random 
- Math
- Uuid

## Fablab Adventure Features 

###### Movement
Players move their sprit using WASD or the arrow keys on a keyboard. 

## How to run 
Run main.py
